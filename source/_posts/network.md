## HTTP

### 常见HTTP返回状态
1、一览表

|  状态集  |       状态码      |
| :-----: | :--------------: |
|   1xx   | 请求收到，继续处理  |
|   2xx   | 操作成功收到，分析、接受 |
|   3xx   | 完成此请求必须进一步处理 |
|   4xx   | 请求包含一个错误语法或不能完成 |
|   5xx   | 服务器执行一个完全有效请求失败 |
